Exercise tracker

Create a New User

POST /api/users

Add exercises

POST /api/users/:_id/exercises

GET user's exercise log: GET /api/users/:_id/logs?[from][&to][&limit]

[ ] = optional

from, to = dates (yyyy-mm-dd); limit = number

Funcionalidades

  • 1 - Create New User Form
  • 2 - Ver TODOS los usuarios /api/users
  • 3 - Add Excercises
  • 4 - ver Todos los logs de UN usuario /api/users/:_id/logs
  • 5 - ver log con FROM query /api/users/:_id/logs?from
  • 6 - ver log con FROM TO query /api/users/:_id/logs?from&to
  • 7 - ver log con FROM TO LIMIT query /api/users/:_id/logs?from&to&limit
